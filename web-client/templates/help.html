Application Help

This page explains the main navigation items. Each menu opens a set of tools for managing your network inventory and configurations.

Inventory Menu

  • Devices – Lists all registered hardware. Select Inventory then Devices to view the table of devices.
  • Show Pad – Provides quick links for consumable orders and inventory pages. Accessed via Inventory > Show Pad.
  • Reports – Summary reports including duplicate checks and consumables. Open from Inventory > Reports.
  • Inventory Settings – Links for editing tags, device types and related settings. Choose Inventory > Inventory Settings.

Network Menu

  • Dashboard – Displays widgets for the selected site. You can pick the site in the dropdown at the top of the dashboard page.
  • Conf t – Configuration tasks such as port or VLAN updates. Links appear in the grid when you select Network > Conf t.
  • Show – Readâ€‘only utilities like port search and IP search. For example, Network > Show > IP Search shows an input box labeled "Enter IP" at the top of the page.
  • Tasks – Access the scheduler and task queue. Navigate to Network > Tasks then choose the desired function.
  • Network Settings – Management pages for VLANs, SSH credentials and SNMP settings. Start from Network > Network Settings.

Admin Menu

Admins and super admins see additional options:

  • My Profile – Update your user preferences.
  • System / Organisation Settings – Configure system wide values and organisational details.
  • Sync / API's – Manage cloud synchronisation and API keys.
  • Logs or Locations / SSH Credentials – Tools specific to your role.
